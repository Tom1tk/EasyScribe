# Transcriber7 - Python dependencies
#
# IMPORTANT: Install PyTorch separately before the other packages.
# Use one of the commands below (copy into PowerShell or cmd):
#
#   CPU only (smaller download, slower transcription):
#     pip install torch torchaudio --index-url https://download.pytorch.org/whl/cpu
#
#   NVIDIA GPU — CUDA 11.8:
#     pip install torch torchaudio --index-url https://download.pytorch.org/whl/cu118
#
#   NVIDIA GPU — CUDA 12.1:
#     pip install torch torchaudio --index-url https://download.pytorch.org/whl/cu121
#
#   NVIDIA GPU — CUDA 12.4 (latest):
#     pip install torch torchaudio --index-url https://download.pytorch.org/whl/cu124
#
# PowerShell warning: NEVER write "torch>=2.3.0" unquoted in PowerShell — the >
# character is a redirect operator and will silently break the install.
# Always quote: pip install "torch>=2.3.0" ...  or just omit the version pin.
#
# The build script (build_windows.bat) handles torch installation automatically.
# Pass --cuda 11.8 / --cuda 12.1 / --cuda 12.4 to select a CUDA build.
